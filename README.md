# tool-type-instead-of-paste

仮想デスクトップなど、コピー＆ペーストができない環境においてテキストを入力してくれるツールです。

## 機能・画面

- 左下：テキスト入力フォーム
- 左上：入力開始ボタン
  - ボタンを押すと3秒後にテキスト入力フォームに記載されたテキストがキーボード入力される
- 右半分：入力履歴
  - それまでに入力したテキストが履歴として表示される
  - 履歴をクリックすると、再度テキスト入力フォームへ入力される
- 右下：入力履歴のクリアボタン
  - 入力履歴をクリアする

## 開発環境

- WPFアプリケーション（.NET）
